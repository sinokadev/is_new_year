# is new year? - Python

This project checks whether a date is January 1st for specific formats. We're adding support for one new format each day until January 1st, 2026. 

# Support Format
 - Korean (added on 2025-12-12)
 - Number (added on 2025-12-13) - only Unix timestamps
 - English (added on 2025-12-14)
 - Date String (planned for 2025-12-15) - String-based date formats: YYYYMMDD (numeric strings) and YYYY-MM-DD with delimiters (space, /, -, ., _) including ISO 8601. For more details, please refer to the [Date String](./Data-String.md) below.
